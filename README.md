## ✨ Portfolio Sections

### 🌟 Hero Section
Professional introduction with animated tagline, developer role title, and smooth entrance effects — built to make a strong first impression.

### 🧑‍🎓 About
Personal academic story, passion for technology, career goals, and creative identity as an engineering student and AI/ML enthusiast.

### 🎓 Education
Clean visual layout presenting degree details, university information, and academic journey.

### 💼 Experience
Vertical animated timeline showcasing roles, responsibilities, and career milestones with glowing markers and smooth transitions.

### 🛠️ Skills
Comprehensive technical stack organized by category — programming languages, frameworks, tools, and technologies.

### 📂 Projects
Featured project cards with:
- Project descriptions
- Live demo links
- GitHub repository buttons
- Hover animation effects

### 🏆 Achievements
College recognitions, technical accomplishments, and extracurricular highlights.

### 🖼️ Gallery
Real activity photos with hover zoom animations — showcasing participation in events, hackathons, and campus activities.

### 📩 Contact
Quick access to all social profiles and a direct contact section for recruiters and collaborators.
